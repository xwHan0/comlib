Process Function List (PFL)

Process Function List provides a probility of using a process function list to replace a process function.

For a function who need a proc argument like:

func(proc, ...)

when the proc argument supports PFL, that means we can use a list [proc1, proc2, ..., procN] to replace proc argument. In this case, the final process result is:

[func(proc1,...), func(proc2,...), ..., func(procN, ...)]
