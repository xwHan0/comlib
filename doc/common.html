Process Function List (PFL)

Process Function List provides a probility of using a process function list to replace a process function.

For a function who need a proc argument like:

func(proc, ...)

when the proc argument supports PFL, that means we can use a list [proc1, proc2, ..., procN] to replace proc argument. In this case, the final process result is an iterator of:

[func(proc1,...), func(proc2,...), ..., func(procN, ...)]

Mininum Data Set (MDS)

Delay Execute (DE)

对于一个高阶函数：

func(proc, ...)

若没有提供足够的有效数据，则该高阶函数返回一个延时函数(假设其名为delay_func)。该延时函数接收原func剩余的有效数据，然后返回最终的执行结果。

Condition Process Filter (CPF)

Process Arguments Map (PAM)

提供对proc和when处理参数的映射功能。一般使用args参数来提供该功能。
